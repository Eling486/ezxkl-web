#ezxkl-web
##——一个较为完善的下课铃投稿系统
</br>
##必要的前置安装
* `Node.js`
* `npm`
	* 因为网络原因，国内推荐使用淘宝镜像：`cnpm`
* `mongoDB`

##安装方式
在根目录下执行：
```
npm install
```
或
```
cnpm install
```
